How it works

Research turned into a habit — the fun part of investing, before any money is on the line.

Run your own in a minute

It's one Python app. Demo mode needs no keys and no money.

# clone & run the safe demo
git clone https://github.com/ASHISH-KUMAR-PANDEY/stocker.git
cd stocker
python3 -m pip install -r requirements.txt
python3 server.py   # → http://127.0.0.1:4173

The fine print

Stocker isn't a brokerage and never holds your money or securities — you self-host and trade through your own broker, so your trades and any losses are your own. The hosted demo is fully simulated. No fractional shares in India — the smallest real buy is one whole share. Company info is illustrative; prices, charts and news are real. Not investment advice; comply with your broker's terms and local regulations.
